# 📚 초등 학년별 주간 시간표 고정 및 자동 배치 시스템
### 💡 스마트한 시간표 관리, 이제 더욱 간편하게!

**학년별 주간 시간표 고정 및 자동 배치 시스템**은 초등학교 교사들을 위한 전문적인 시간표 관리 솔루션입니다. 복잡하고 번거로운 시간표 작성 작업을 자동화하고 체계적으로 관리할 수 있도록 도와드립니다.

---

### ✨ 핵심 기능

#### 1️⃣ **연간시수표 자동 관리**
- 📊 CSV 파일로 연간시수표를 한 번에 불러오기
- 📅 1학기/2학기 구분 관리
- 🎯 주별 목표시수 자동 설정
- 📈 수업일수 및 기간 정보 관리

#### 2️⃣ **고정 교과 시간표 설정**
- 👨‍🏫 전담 교사 시간표를 한눈에 관리
- 🎨 과목별 색상 구분으로 시각적 편의성 제공
- 🔄 드래그 앤 드롭으로 간편한 배치
- 📋 여러 학급에 동시 적용 가능

#### 3️⃣ **스마트 자동 채우기**
- 🤖 목표시수를 기반으로 자동 시간표 생성
- ⭐ 과목별 선호교시 설정 지원
- 🚫 비활성화된 요일/교시 자동 제외
- ⚡ 몇 초 만에 완벽한 시간표 완성

#### 4️⃣ **주별 시간표 관리**
- 📆 학기별 주차별 독립적인 시간표 관리
- 💾 각 주의 데이터 자동 저장 및 불러오기
- 🔄 주 간 데이터 복사 기능
- 📊 주별 달성도 실시간 확인

#### 5️⃣ **실시간 달성도 모니터링**
- 📊 전체 학급 주간 시수 및 달성도 통합 표시
- ✅ 목표시수 대비 현재 시수 자동 계산
- 🎨 색상으로 달성도 시각화
- 📈 과목별 진행 상황 한눈에 파악

#### 6️⃣ **유연한 설정 옵션**
- 📅 요일별 비활성화 기능 (주별 독립 적용)
- ⏰ 요일별 수업 수 제한 설정
- 🎯 과목별 선호교시 설정
- 👥 학급 수 및 교시 수 자유롭게 조정

#### 7️⃣ **엑셀 연동**
- 📥 엑셀 파일로 시간표 내보내기
- 📤 엑셀 파일에서 시간표 불러오기
- 📋 통합된 시간표 시트 생성
- 💼 문서화 및 공유 용이

#### 8️⃣ **안전한 데이터 관리**
- ☁️ 서버 자동 저장
- 💾 로컬 백업 기능
- 🔄 데이터 복원 지원
- 🔒 계정별 독립적인 데이터 관리

## 📖 이용 방법 가이드

### 🚀 시작하기
접속 주소 https://timetable-2cvy.onrender.com/

#### 1단계: 계정 만들기
1. 로그인 페이지에서 **"계정 만들기"** 버튼 클릭
2. 학교명, 학년, 닉네임, 비밀번호 입력
3. 계정 생성 완료 후 자동 로그인

#### 2단계: 연간시수표 입력
1. 메인 페이지에서 **"연간시수표"** 버튼 클릭
2. **"예시파일"** 다운로드하여 형식 확인
3. 이지 에듀 시수표 CSV 파일 선택 또는 직접 입력
4. 1학기/2학기 구분하여 저장
5. **"저장"** 버튼 클릭

#### 3단계: 연간시수표 로드
1. 메인 페이지로 돌아가기
2. 학기 선택 (1학기/2학기)
3. **"로드"** 버튼 클릭하여 연간시수표 불러오기
4. 첫 번째 주가 자동으로 선택됨

---

### 📝 시간표 작성하기

#### 고정 교과 설정
1. **"고정 교과 및 전담 교사 설정"** 섹션에서
2. **"과목 추가"** 버튼으로 과목 선택
3. 시간표에서 해당 시간대 클릭
4. 적용할 학급 선택 (여러 학급 동시 선택 가능)
5. 드래그 앤 드롭으로 빠르게 복사 가능

#### 자동 채우기 사용
1. 각 학급 카드에서 **"자동 채우기"** 버튼 클릭
2. 목표시수를 기반으로 자동 배치
3. 선호교시 설정이 있으면 우선 적용
4. 비활성화된 요일/교시는 자동 제외

#### 수동 입력
1. 시간표 셀 클릭
2. 과목 선택 팝업에서 원하는 과목 선택
3. 드래그 앤 드롭으로 이동 가능

---

### ⚙️ 고급 기능 활용하기

#### 요일 비활성화
1. 시간표 헤더의 요일명(월, 화, 수, 목, 금) 클릭
2. 해당 요일 열이 비활성화됨 (현재 주에만 적용)
3. 다시 클릭하면 활성화

#### 목표시수 설정
1. **"전체 학급 주간 시수 및 달성도"** 표에서
2. 목표시수 셀 클릭하여 값 입력
3. 모든 학급에 동일하게 적용됨
4. 달성도가 자동으로 계산되어 표시됨

#### 주 변경하기
1. 화살표 버튼(◀ ▶)으로 주 이동
2. 각 주의 데이터는 자동으로 저장/불러오기
3. 주별로 독립적인 시간표 관리

#### 관리자 설정
1. 메인 페이지에서 **"관리자"** 버튼 클릭
2. **학급 수 설정**: 사용할 학급 수 변경
3. **교시 수 설정**: 하루 교시 수 변경
4. **과목별 선호교시 설정**: 자동 채우기 시 우선 적용
5. **요일별 수업 수 설정**: 특정 요일의 수업 수 제한

---

### 💾 데이터 관리

#### 서버에 저장
- **"서버에 저장"** 버튼 클릭
- 모든 변경사항이 서버에 저장됨
- 다른 기기에서도 동일한 데이터 접근 가능

#### 엑셀로 내보내기
1. **"엑셀 내보내기"** 버튼 클릭
2. 고정교과 시트와 통합 시간표 시트 생성
3. 모든 학급의 시간표가 하나의 파일에 저장

#### 엑셀에서 불러오기
1. **"엑셀 불러오기"** 버튼 클릭
2. 이전에 내보낸 엑셀 파일 선택
3. 시간표 데이터 자동 복원

#### 백업 및 복원
1. 관리자 페이지에서 **"전체 데이터 백업"** 클릭
2. JSON 파일로 다운로드
3. 필요 시 **"백업 불러오기"**로 복원

---

### 🎯 효율적인 활용 팁

1. **연간시수표 먼저 입력**: 시간표 작성 전에 반드시 연간시수표를 입력하세요.
2. **고정 교과 우선 설정**: 전담 교사 시간표를 먼저 설정하면 나머지 작업이 쉬워집니다.
3. **자동 채우기 활용**: 대부분의 시간표는 자동 채우기로 생성한 후 수정하는 것이 빠릅니다.
4. **주별 독립 관리**: 각 주의 특성에 맞게 요일 비활성화 등을 활용하세요.
5. **달성도 확인**: 주간 시수 표를 통해 목표시수 달성 여부를 실시간으로 확인하세요.

### ❓ 자주 묻는 질문

**Q: 여러 학급의 시간표를 동시에 관리할 수 있나요?**  
A: 네, 학급 수를 설정하면 모든 학급의 시간표를 한 화면에서 관리할 수 있습니다.

**Q: 주별로 다른 시간표를 만들 수 있나요?**  
A: 네, 각 주는 독립적으로 관리되며 요일 비활성화 등도 주별로 적용됩니다.

**Q: 자동 채우기 결과를 수정할 수 있나요?**  
A: 네, 자동 채우기 후 언제든지 수동으로 수정하거나 드래그 앤 드롭으로 이동할 수 있습니다.

**Q: 데이터는 어디에 저장되나요?**  
A: 서버에 저장되며, 로컬 브라우저에도 백업됩니다. 계정별로 독립적으로 관리됩니다.

**Q: 엑셀 파일 형식은 어떻게 되나요?**  
A: 고정교과 시트와 통합 시간표 시트로 구성되며, 표준 엑셀 형식입니다.
